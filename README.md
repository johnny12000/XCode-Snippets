# XCode-Snippets
Useful snippets for XCode

## Setup

1. Clone repository to your CodeSnippets folder: ~/Library/Developer/Xcode/UserData/CodeSnippets

```console
git clone https://github.com/johnny12000/XCode-Snippets.git CodeSnippets
```
